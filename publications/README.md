# Publications Directory

This directory contains all generated inventions organized by domain and date.

## Structure

```
publications/
├── mechanical-engineering/
│   └── YYYY-MM-DD/
│       └── inv-YYYYMMDD-HHMMSS.md
├── materials-science/
├── chemical-engineering/
├── pharmaceutical-chemistry/
├── electrical-engineering/
├── software-algorithms/
├── biotechnology/
├── environmental-technology/
├── medical-devices/
├── agricultural-technology/
└── index.json
```

## File Format

Each invention is stored as a Markdown file with:
- Frontmatter metadata (title, domain, date, ID, hash)
- Title and domain classification
- Technical description generated by AI
- Publication timestamp
- Verification hash

## Index

The `index.json` file maintains a master index of all publications with their IDs, domains, timestamps, and hashes.
